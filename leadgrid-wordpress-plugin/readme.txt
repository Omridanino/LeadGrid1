Advanced integration between LeadGrid landing pages and WordPress with full editing capabilities and real-time synchronization.

== Description ==

LeadGrid Integration Pro allows you to seamlessly import your professionally designed landing pages from LeadGrid directly into your WordPress website. With full editing capabilities, responsive design, and active contact forms, your pages will look and function perfectly.

= Key Features =

* **One-Click Import** - Import your LeadGrid pages directly to WordPress
* **Responsive Design** - All pages are automatically optimized for mobile and desktop
* **Active Contact Forms** - Contact forms work out of the box
* **SEO Optimized** - Pages include proper meta tags and SEO optimization
* **Full Hebrew Support** - Complete RTL support for Hebrew websites
* **Easy Editing** - Edit imported pages using WordPress editor
* **Professional Templates** - Beautiful, conversion-optimized designs

= How It Works =

1. Create your landing page at LeadGrid.co.il
2. Get your API credentials from the publish page
3. Install and activate this plugin
4. Enter your API credentials in the settings
5. Import your page with one click
6. Customize and edit as needed in WordPress

= Requirements =

* WordPress 5.0 or higher
* PHP 7.4 or higher
* Active LeadGrid account with created landing page

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/leadgrid-integration` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to LeadGrid > Settings to configure your API credentials
4. Import your pages from LeadGrid > Import Pages

== Frequently Asked Questions ==

= Do I need a LeadGrid account? =

Yes, you need to create a landing page at LeadGrid.co.il first to get the API credentials.

= Are the contact forms functional? =

Yes, all contact forms are fully functional and ready to receive submissions.

= Can I edit the imported pages? =

Absolutely! Once imported, you can edit the pages using the WordPress editor.

= Is it mobile responsive? =

Yes, all imported pages are fully responsive and optimized for all devices.
